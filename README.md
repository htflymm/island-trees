# 神木之島 民間支援站

紀錄片《神木之島 The Island and the Trees》民間自發整理的支援頁面。

**網站：** https://htflymm.github.io/island-trees/

## 為什麼

紀錄片《神木之島》於 2026/5/15 全台上映。導演李香秀辭去 23 年副教授職位、用退休金自費製作五年，但發行行銷不足，首映日有戲院票房掛零。

本 repo 是民間影迷自發整理的支援站，目的是：

- 整理全台訂票管道，降低觀眾進戲院的摩擦
- 彙整延伸閱讀，便於認識電影與背後團隊
- 提供社群可分享的入口，協助第一週票房

## 給官方團隊

內容歡迎自由使用、轉載、修改。如想接手此頁面：

- **Fork 這個 repo**，修改 footer 後部署到官方網域
- 或開 **GitHub Issue** 聯繫 ownership 轉移

## 內容來源

- 訂票連結：各院線官方頁面
- 延伸閱讀：環境資訊中心、雪羊隨筆（vocus）、中央社、中時新聞網
- 預告片：官方 YouTube 嵌入
- 引言：李香秀導演公開受訪內容
